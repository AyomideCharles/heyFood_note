# heyfood_note_app

A note app that allows users to create, edit, view and delete notes.

# Set up instructions
- git clone https://github.com/AyomideCharles/kobipay_test.git
- cd transactions-app
- flutter pub get (to install dependencies)
- flutter run

# Features
- Add note
- Delete note
- Edit note
- Search note
- Filter note by categories
- Add new categories
- Filter notes by date added
- Filter notes by alphabetical orders
- Dark mode and light mode


# Libaries used
- GetX - State management
- Flutter screenutil
- get storage


# Architecture

lib/ 
├── core/
├── models/
├── controller/
├── services/
├── widgets/
└── screens/
